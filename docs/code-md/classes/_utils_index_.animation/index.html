    Class: Animation

    Provides the ability to interpolate and schedule events over a given range. Intended to help animate parts of a page not accessible through CSS such as the scroll bar.

    var animation = new Animation(AnimationType.EaseIn, 500, (next) => {
 this.state = { scrollTop: next };
});
animation.Animate(0, 100);
